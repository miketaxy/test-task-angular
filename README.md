Тестове завдання на роботу для контролювання заборгованістю
# Опис програми

Цей проект є прикладом фінансового дашборду для контролювання заборгованістю.

## Як запустити проект

1. Встановіть необхідні залежності:
    ```bash
    npm install
    ```

2. Запустіть сервер розробки:
    ```bash
    ng serve
    ```

3. Відкрийте браузер і перейдіть за адресою `http://localhost:4200/`.

## Структура проекту

- `src/` - вихідний код додатку
- `app/` - основні компоненти та модулі додатку
- `assets/` - статичні файли та ресурси
- `environments/` - конфігурації середовищ

## Вимоги

- Node.js
- Angular CLI
